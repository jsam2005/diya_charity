# Logos Directory

This directory contains the certification logos for the website.

## Required Logo Files

Please add the following logo images to this directory:

1. **ngo-npo-logo.png** - Logo for NGO & NPO Registration
   - Recommended size: 200x200px or larger
   - Format: PNG with transparent background (preferred)
   - Used in: NGO & NPO Registration certification box

2. **darpan-logo.png** - Logo for DARPAN Portal Registration
   - Recommended size: 200x200px or larger
   - Format: PNG with transparent background (preferred)
   - Used in: DARPAN Portal Registration certification box

3. **income-tax-logo.png** - Logo for Income Tax Registration
   - Recommended size: 200x200px or larger
   - Format: PNG with transparent background (preferred)
   - Used in: Income Tax Registration certification box

## Supported Formats

- PNG (recommended for transparency)
- JPG/JPEG
- SVG (scalable vector graphics)
- WebP (modern format with good compression)

## Notes

- Logos will be automatically resized to fit the certification boxes
- If a logo file is missing, the image will be hidden gracefully
- Ensure logos have good contrast against the transparent gradient background
